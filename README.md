# Scarlet Dress Shop Database System
MySQL and Python database system for managing customers, orders, products, and tailoring services


A database management system designed for a bridal and formalwear shop using MySQL and Python.

## Overview
This project manages the main operations of a dress shop, including customers, employees, suppliers, products, orders, payments, and tailoring services.

## Features
- Customer and employee management
- Product and supplier management
- Orders and order items tracking
- Payment records
- Tailoring services and service requests
- Relational database design with proper normalization

## Database Design
The system includes multiple related entities such as:
- Customers
- Employees
- Suppliers
- Products
- Orders
- Order Items
- Payments
- Tailoring Services
- Tailor Service Requests

## Technical Details
- Database: MySQL
- Backend: Python
- Frontend: HTML, CSS, JavaScript

## Highlights
- ERD design
- One-to-many and many-to-many relationships
- Database normalization up to 3NF
- Scalable structure for future web or POS integration


This repository includes the database project files, SQL setup, and project report.
